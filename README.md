# 🍳 ServD - AI Recipes (SaaS)

ServD AI Recipes - A full-stack AI-powered recipe platform that helps users create delicious recipes based on available ingredients, search for any recipe, and save their favorites for later. Built with modern web technologies and a subscription-based model.

> ✨ **Create, search, and save AI-powered recipes using ingredients you already have.**

## ✨ Features

- 🧠 AI Recipe Generation using Gemini API  
- 🔍 Search any recipe instantly  
- 💾 Save favorite recipes  
- 🥕 Pantry management system  
- 📸 AI image-based ingredient detection  
- 🖼️ Recipe images via Unsplash API  
- 🔐 Authentication with Clerk  
- 💳 Subscription system (Free & Pro)

## 🚀 Getting Started

```bash
git clone https://github.com/asifcuber08/ServD--Ai-Recipes.git
cd ServD--Ai-Recipes

```

## ⚙️ Backend Setup
``` bash
cd backend
npm install
npm run develop
```

---

## 🎨 Frontend Setup
``` bash

```
